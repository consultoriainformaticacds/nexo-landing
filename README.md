# nexo-landing

Multi-model landing page showcases and sales materials for web design/web presence services.

## Tech Stack
- HTML5
- CSS3 (embedded)
- JavaScript (inline)
- Google Fonts (various per demo)

## Setup
Open index.html or any demo-*.html in a web browser to view the pages.

## Structure
- index.html: Main page
- demo-*.html: Aesthetic and store demos (premium/standard)
- comparativa/index.html: Comparison table
- modelos/*.html: Industry-specific templates (barber shop, accounting, legal, psychologist, restaurant)
- assets/pdf/: PDF assets
