# 🚀 CDS & NEXO - Ecosistema Digital 2026

Este repositorio (o grupo de repositorios) conforma la infraestructura digital de **CDS Consultoría Informática** y su división de diseño **NEXO WEB**.

## 🏗️ Estructura del Proyecto

El ecosistema se divide en 3 pilares fundamentales conectados entre sí:

### 1. 🛡️ CDS - Consultoría Informática (Matriz)
*   **URL:** [https://consultoriainformaticacds.pages.dev/](https://consultoriainformaticacds.pages.dev/)
*   **Función:** Generación de autoridad, servicios de ciberseguridad, soporte IT y cimientos técnicos.
*   **Stack:** HTML5, CSS3 (Elite UI), JavaScript (IntersectionObserver).

### 2. 🎨 Nexo Web (Agencia Boutique)
*   **URL:** [https://nexoweb.pages.dev/](https://nexoweb.pages.dev/)
*   **Función:** Página de venta de servicios de diseño y desarrollo. Enfoque en conversión y psicología de ventas High-Ticket.
*   **Componentes Clave:** Ticker dinámico, Matriz de servicios, Tabla de inversión.

### 3. 📁 Nexo Landing (Catálogo y Demos)
*   **URL:** [https://nexo-landing.pages.dev/](https://nexo-landing.pages.dev/)
*   **Función:** Showroom interactivo para que el cliente experimente las arquitecturas antes de contratarlas.
*   **Módulos:** /assets/comparativa (Guía de decisión para el cliente).

## 🛠️ Stack Tecnológico Común
*   **Frontend:** HTML5 Semántico, CSS3 con metodología de variables (Root) y efectos Glassmorphism.
*   **Deploy:** Cloudflare Pages (CI/CD automático desde GitHub).
*   **Conversión:** Formspree API (Formularios AJAX) y WhatsApp Business integration.

## 📈 Flujo de Conversión (Funnel)
1. **Atención:** Publicidad o SEO directo a CDS o Nexo Web.
2. **Interés:** Navegación por el catálogo de demos en Nexo Landing.
3. **Deseo:** Consulta de la Comparativa de Modelos para entender el valor.
4. **Acción:** Solicitud de Diagnóstico Gratuito vía Formulario o WhatsApp.
